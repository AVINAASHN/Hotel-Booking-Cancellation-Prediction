Developed a Hotel Booking Cancellation Prediction system using Machine Learning to identify reservations likely to be canceled. Performed data cleaning, feature engineering, missing value handling, categorical encoding, and model training on hotel booking data. Built an end-to-end Scikit-Learn pipeline and achieved 88.02% accuracy with a ROC-AUC score of 0.95 using Random Forest. Serialized the trained model with Joblib and prepared it for deployment through a Streamlit web application.
